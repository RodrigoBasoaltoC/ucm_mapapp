# UCM Mapp App

Esta app permite visualizar y localizar sectores y salas dentro del campus de la Universidad Católica del Maule.

## Tecnologías
- Flutter
- Dart
- MySQL
- Flask (API backend)

## Requisitos
- Flutter SDK 3.x o superior.
- Android Studio / Visual Studio Code configurado con el plugin de Flutter y Dart.
- Un **emulador Android** o un **dispositivo físico** conectado por USB
- Node.js

### Pasos para ejecutar

1. **Clonar este repositorio:**
   ```bash
   git clone https://github.com/RodrigoBasoaltoC/ucm_mapapp.git
   
2. Abre Android Studio y selecciona Open an existing project y elige la carpeta clonada (ucm_mapapp)

3. Instalar las dependencias con flutter get

4. Ejecutar la aplicación con flutter run


Una vez abierta la app, toca el ícono de edificio y se mostrará la lista de pisos y salas simuladas, representando la estructura del campus.
